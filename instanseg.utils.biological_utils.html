instanseg.utils.biological_utils.resolve_cell_and_nucleus_boundaries(lab: Tensor, allow_unnucleated_cells: bool = True) → Tensor

lab: tensor of shape 1,2,H,W containing nucleus and cell labels respectively

returns: tensor of the same shape as lab

This function will resolve the boundaries between cells and nuclei. It will first match the labels of the largest nucleus and its cell. It will then erase from the cell masks all the nuclei pixels. This resolves nuclei “just” overlapping adjacent cell. It will then recover the nuclei pixels that were erased by adding them back to the cell masks.

allow_unnucleated_cells: If False, this will remove all cells that don’t have a nucleus.
