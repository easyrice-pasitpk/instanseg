instanseg.utils.utils.save_image_with_label_overlay(im: ndarray, lab: ndarray, output_dir: str = './', base_name: str = 'image', clip_percentile: float = 1.0, scale_per_channel: bool | None = None, label_boundary_mode='inner', label_colors=None, alpha=1.0, thickness=3, return_image=False)ÔÉÅ

Save an image as RGB alongside a corresponding label overlay. This can be used to quickly visualize the results of a segmentation, generally using the default image viewer of the operating system.

Parameters:

  • im ‚Äì Image to save. If this is an 8-bit RGB image (channels-last) it will be used as-is, otherwise it will be converted to RGB

  • lab ‚Äì Label image to overlay

  • output_dir ‚Äì Directory to save to

  • base_name ‚Äì Base name for the image files to save

  • clip_percentile ‚Äì Percentile to clip the image at. Used during RGB conversion, if needed.

  • scale_per_channel ‚Äì Whether to scale each channel independently during RGB conversion.

  • label_boundary_mode ‚Äì A boundary mode compatible with scikit-image find_boundaries; one of ‚Äòthick‚Äô, ‚Äòinner‚Äô, ‚Äòouter‚Äô, ‚Äòsubpixel‚Äô. If None, the lab is used directly.

  • alpha ‚Äì Alpha value for the underlying image when using the overlay. Setting this less than 1 can help the overlay stand out more prominently.

Returns:

The input image with corresponding label overlay if lab is a torch Tensor, or nothing otherwise.
